HTML5 - Шаблон для разработки

Что использует шаблон

  • Gulp - сборщик проекта
  • Pug - вместо html
  • Sass/SCSS - вместо css
  • Grid System - модульная сетка
  • Fontawesome - иконки сайта

Готовые миксины

  • +font | @include font - для подключения шрифтов
  • +margin-v | @include margin-v - для создания вертикальных отступов
  • +margin-h | @include margin-h - для создания горизонтальных отступов
  • +placeholders | @include placeholders - для быстрой стилизации плейсхолдеров под все браузеры
  • +before('content') | @include before('content') - замена стандартному ::before
  • +after('content') | @include after('content') - замена стандартному ::after
  • +first | @include first - изменение первого элемента
  • +last | @include last - изменение последнего элемента
  • +nth | @include nth - изменение элемента по счету

Виды изображений:

  • .bg(data-src="Путь") - задний фон на всю высоту и длину блока
  • .lazy(data-src="Путь") - ленивая загрузка
  • .svg(data-src="Путь") - смена цвета svg

Навигация по шаблону:

/ - Корень
/app/ - папка разработки
/app/img - папка не сжатых изображений
/app/js - папка javascript'a для разработки
/app/pages/layouts - шаблоны (Не компилируются)
/app/pages/*.pug - страницы сайта для разработки
/app/sass/utils - разбивание стилей по файлам (Не компилируются)
/app/sass/_*.sass стили страницы (Не компилируются)
/app/sass/*.sass - стили страницы для разработки
/build/ - папка собранного проекта
/build/css - готовые стили
/build/fonts - шрифты сайта
/build/img/ - папка сжатых изображений
/build/img/icons/ - папка иконок
/build/js - папка javascript'a
/build/*.html - страницы сайта
/node_modules - модули node.js
/.browserlistrc - для модуля gulp-autoprefixer
/gulpfile.js - сборщик проекта
/package.json - инфо о всем
